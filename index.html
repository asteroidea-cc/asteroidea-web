SIA Premises Equipment (PE) Tester for Arduino R4 WiFi

A simple test tool built on Arduino for sending alarm messages using the SIA-DCS protocol over TCP/IP.

This tool is designed for developers and integrators who need to simulate and validate alarm transmissions to cloud-based receivers. It enables full customization of SIA message fields and provides real-time monitoring and feedback through the Serial interface.

Key Features
  • Connects to Wi-Fi and resolves the address of a cloud-based alarm platform
  • Allows manual configuration of:
    • Sequence number
    • Receiver identifier
    • Location prefix
    • Account identifier
  • Generates valid SIA-DCS messages including:
    • Framing with 0x0A and 0x0D
    • Payload length in hexadecimal
    • CRC-16 checksum calculated using a compact implementation
  • Sends structured messages over TCP and waits for ACK or NAK responses
  • Provides real-time feedback via the Serial Monitor
Code Input Modes
  • Direct SIA code entry (e.g. BA for Burglary)
  • Input of classification,alarmcode pairs, mapped to SIA codes using a built-in reference with over 200 predefined alarm types
