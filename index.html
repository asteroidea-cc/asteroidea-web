Projects

Arduino SIA Tester

Simple SIA-compatible test tool built on Arduino for sending alarm messages to cloud-based alarm platforms using the SIA-DCS protocol over TCP/IP.

It connects to Wi-Fi, allows manual configuration of alarm fields (like sequence, receiver, location prefix, and account), and sends structured messages through a secure socket. The system waits for ACK or NAK responses and supports retry logic and real-time feedback via Serial Monitor.

Ideal for testing and validating SIA integration endpoints in a controlled lab environment.
