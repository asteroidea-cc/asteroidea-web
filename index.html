SIA Premises Equipment (PE) Tester

A quick and simple tool for simulating and validating SIA-DCS alarm messages over TCP/IP. Designed for testing towards cloud-based alarm receivers, the solution provides control over SIA message structure and transmission.

Developed for the Arduino R4 WiFi platform

No Encryption Support

Key Features
  • Establishes Wi-Fi connectivity and resolves DNS for target alarm servers
  • Supports manual configuration of:
    • Sequence number
    • Receiver ID
    • Location prefix
    • Account identifier
  • Constructs compliant SIA-DCS messages with:
    • Framing using 0x0A and 0x0D
    • Hexadecimal payload length
    • CRC-16 checksum via compact algorithm
  • Sends structured messages over TCP and handles ACK/NAK responses
  • Offers real-time feedback through the Serial Monitor
Input Methods
  • Direct entry of two-character SIA codes (e.g., BA for Burglary)
  • Classification-based entry using classification,alarmcode format, mapped to over 200 predefined alarm types
